# Git Status Skill

This skill allows the agent to inspect the current Git status, identify the active branch, and view all registered git worktrees in the project.

## Execution

To run this skill and display status, execute the following script from the root of the repository:

```bash
node .agents/skills/git-status/git_status.js
```
